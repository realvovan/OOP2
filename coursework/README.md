# OOP Coursework

Realtor manager program - a winforms application that lets you:
* Edit real estate
* Set its price and other parameters
* Edit clients
* Set their preferred price and room count
* Add suggested real estate based on clients' preferences
* Sort and search entities by their parameters
* Save and load entites using Json

UI made using Winforms<br>
